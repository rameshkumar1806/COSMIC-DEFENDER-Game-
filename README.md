Cosmic Defender is an interactive 2D space shooter game developed in Python. The player controls a spaceship, destroys enemy waves, and tracks performance through a live analytics dashboard.

The project uniquely combines:

🎮 Pygame for core game mechanics
🎬 Turtle for intro animation
📊 Matplotlib for real-time stats visualization
🔥 Features
Player movement and shooting system
Multiple enemy waves with increasing difficulty
Explosion particle effects
Animated star background
Live dashboard showing:
Score progression
Accuracy percentage
Wave level
Kill heatmap
Pause and restart functionality
🎮 Controls
Left / Right Arrow → Move
Space → Shoot
P → Pause
R → Restart
ESC → Quit
